# Minecraft-King-Client
This Is A Minecraft Hack Menu Client. There are some commands already built in there but the rest is for your creative mind to make. You have to make your script in Python But There will be a Java Version Soon. Please Read The Read Me If You Want To Know How To Execute Your Minecraft Scripts

How To Use To Execute Your Own Script:

Put your Python Minecraft script in the "scripts" folder. The click your python script do "CTRL + C" Or right click it and press "Copy" The go Into Terminal And Paste It In With "CTRL + V", Then copy that text you just pasted. Then go into The Minecraft King Client And Inside Of The TextBox/Executor Box Paste It In The Click "Execute'
